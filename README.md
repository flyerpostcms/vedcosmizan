# VedCosmizan

**AI-powered Vedic Astrology + Numerology app, in 5 languages, with PDF reports.**

Built with Flutter (Dart) + Supabase + free AI (Gemini / Groq / Cerebras) + Swiss Ephemeris.

- **Languages:** English, Hindi, Malayalam, Tamil, Arabic (app UI + PDF output)
- **Platforms:** Android (now), iOS (later)
- **Two faces:** Vedic Jyotish (India) and a neutral "Cosmic Blueprint" mode (Gulf)

## What this repo contains

Only the **application code**. It does **not** contain:
- the astrological prediction database (private content, separately copyrighted), or
- any API keys, credentials, or signing keys.

These are listed in `.gitignore` and stay on the developer's machine.
